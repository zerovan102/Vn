## How to run locally

1. Install dependencies
```bash
npm install
```

2. Run dev server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
npm run preview
```

## Notes
- The contact form uses **emailjs-com**. Replace the placeholders in `src/components/Contact.jsx`:
  - YOUR_SERVICE_ID
  - YOUR_TEMPLATE_ID
  - YOUR_PUBLIC_KEY

- You can customize content in `src/components/*` and replace avatar in `public/avatar.jpg`.
